# Starweb Software Development Kit


## System Requirements
- PHP >= 7.1  (extension requirements in composer.json).

## Installation
Use [composer](https://getcomposer.org/?target=_blank) to install the package
```bash
$ composer require starweb/starweb-sdk
```
